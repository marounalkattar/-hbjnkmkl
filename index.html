Embedded system mini project:

This is an Embedded system project at University La Sagesse. The hardware used in this project is shown in the photo.

component used are:

1- Arduino mega 2560 x1
2- Pull up button x4
3- Potentiometer x3
4- Joystick x1
5- Lcd (16,2) x 1
6- RGB x1
7- LED BAR (10 LED) x1
8- Seven segment X2

About this project:

Embedded testing is the process of verification and validation of both software and hardware. It ensures the defect free whole system including software and hardware. It is basically performed on hardware to find the defects. So, the purpose of this project is to test the board before using it. This is a university board, so this project is importin because this board are used from different student and can be broken like every electronic device.

Some explanation of this project

This project is divided in 10 state using state machine, in each state the two seven segments will affiche a number (ex: state 0 the seven segments will affiche 00), a LED will go on from the 10 led bar, the seven segment and the led bar use object oriented programming to set the pin and the number (classes) and a color will be testing in the RGB (state 0 red, state 1 green, state 3 blue), and a INPUT will be tested. To better understand see the YouTube video attached.

This project are tested in proteus before using on the board.
